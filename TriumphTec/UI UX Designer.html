UX/UI Designer

UX é a experiência do usuário, é a relação de uma pessoa com a experiencia de um produto ou serviço, o UX designer elabora mapas de navegação do usuario, pesquisa e proto tipa. Ele é o responsável por fazer os testes de usabilidade e de desenvolver soluções inovadoras
Muito além de um design bonito, quem trabalha com UX Design precisa focar as pessoas.
Entre as muitas perguntas que precisam ser respondidas para alcançar a eficiência da experiência estão, como a pessoa vai usar o produto?
UI é a interface do usuário, é a parte visual e de interação com um determinado produto ou serviço.
Imagine um botão de aceitar na cor vermelha ou um ícone de X para fazer login. Esses são desastres prontos para acontecer! A principal tarefa de um UI Designer é desenhar uma interface de forma que ela seja clara, objetiva e principalmente intuitiva para o usuário.
O papel dele é conhecer padrões de uso da sociedade e costumes subjetivos para tornar o uso de um aparelho eletrônico tão natural que o usuário nem precise pensar.
As áreas de atuação
desenvolvimento de web sites e aplicativos, análise de usabilidade, rquitetura da informação, UX Research, Ux Writer. a média salarial pode variar de R$ 2.000 para Ux Designer Jr. a R$ 8.000,00 para Ux Designer Sênior.
